
# Telegram Coupon Bot

Этот бот дарит 12 цифровых купонов. При выборе купона он исчезает из меню.

## Как запустить:
1. Установи Python 3.10+.
2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Получи токен от @BotFather и вставь в `bot.py` вместо "ТОКЕН_ТВОЕГО_БОТА".
4. Положи картинку с описанием купонов в этот же каталог с именем `coupons.jpg`.
5. Запусти бота:
   ```bash
   python bot.py
   ```
